# BLVQ Customer Balance PWA - Frontend

Progressive Web App for checking shop credit balances.

## Features

- ✅ PWA with **automatic updates** (no cache clearing needed!)
- ✅ Installable to home screen  
- ✅ Works offline
- ✅ Customer balance checking via UUID
- ✅ Admin dashboard for customer linking
- ✅ QR code generation
- ✅ Responsive design
- ✅ Real-time data with caching

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Deployment to GitHub Pages

The app auto-deploys when you push to `main` branch.

### Setup:
1. Enable GitHub Pages (Settings → Pages → Source: GitHub Actions)
2. Push code to GitHub
3. Done! Auto-deploys on every push

### PWA Auto-Update:
When you push updates, users get the latest version automatically within 60 seconds. No cache clearing needed!
